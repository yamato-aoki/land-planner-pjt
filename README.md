# 立地開発プランナー公式サイト

土地活用・相続対策・店舗開発の専門家「立地開発プランナー」公式Webサイト（Next.js + TypeScript + Tailwind CSS）。
高齢地主様や企業担当者向けに、信頼・安心・専門性を伝えるブランドサイトです。

---

## プロジェクト概要

「立地開発プランナー（宅地建物取引士）」の個人名刺代わりとなるプロフェッショナルサイト。
主な目的は、土地活用・相続対策・店舗開発の相談を「LINE」または「電話」で受けること。

## サイトコンセプト

- **ターゲット:** 土地活用に悩む地主、相続対策を考える個人オーナー、出店地を探す企業担当者
- **トーン＆マナー:** 信頼感・誠実さ・プロフェッショナル・親しみやすさ
- **ゴール:** LINEまたは電話での問い合わせ

## ページ構成

1. **トップページ**
    - キャッチコピー：「その土地の未来を、確かな専門知識で拓く。」
    - メインビジュアル（鳥の写真）
    - 4つの主要セクション（プロフィール・ソリューション・仕事の流れ・お問い合わせ）
    - フローティングLINEボタン
2. **プロフィール・実績**
    - 宅地建物取引士資格、所属、メッセージ、実績（数字で信頼を可視化）
3. **ソリューション**
    - 相続税対策、空き地活用、テナント仲介など
4. **仕事の流れ**
    - ヒアリング→調査→提案→契約・実行の4ステップ
5. **お問い合わせ**
    - LINE・電話・メールでの相談窓口

## 技術スタック

- **フロントエンド:** Next.js (App Router), TypeScript, Tailwind CSS, shadcn/ui
- **インフラ:** AWS Amplify Hosting（またはS3+CloudFront）
- **アニメーション:** Framer Motion

## デザインシステム

- **メインカラー:** ディープ・フォレストブルー (#1A2E35)
- **アクセントカラー:** ヘリテージ・ゴールド (#B59A6D)
- **背景:** サンドベージュ (#F9F7F2)
- **サブテキスト:** スレートグレー (#4A5568)
- **フォント:** Noto Serif JP（見出し）、Noto Sans JP（本文）、Cinzel（英語表記）
- **和紙テクスチャ:** CSSで背景に和紙風ノイズを重ね高級感を演出

## 特徴・演出

- Heroセクション：グラデーション＋鳥写真＋キャッチコピー、フェードインアニメーション
- 各セクション：スタッガーアニメーション、カードホバー、カウントアップ演出
- お問い合わせ：LINEボタン常時表示、タップ時のマイクロインタラクション
- レスポンシブ対応（PC/スマホ両対応、SP用画像切替）
